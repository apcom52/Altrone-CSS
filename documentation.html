Кнопки

Стандартный стиль кнопок применяется к тегам <button>, <input type="button">, <input type="submit"> и <input type="reset">, Можно применить класс кнопок к другим тегам, присвоив им класс .button

Модификаторы

Цвета кнопок

.button--color-*, где вместо * нужно подставить определенный цвет (red, orange, yellow, olive, green, teal, blue, violet, purple, pink, brown, black)

Скругление кнопок

Для скругления краев используются классы:

  • .button--smooth - небольшое скругление, чуть больше стандартного
  • .button--rounded - большое скругление краев
  • .button--circle - круглая кнопка
  • .button--no-smooth - полностью убирает скругление

Кнопка с иконкой

Если в кнопке используется шрифтовая иконка, то нужно использовать следующие модификаторы:

  • .button--icon - если в кнопке есть иконка и название
  • .button--icon-only - если в кнопке есть только иконка, без подписи 

Размеры кнопок

  • .button--size-small - маленькая кнопка, на 20% меньше стандартной
  • .button--size-normal - размер кнопки по умолчанию
  • .button--size-large - размер кнопки на 30% больше стандартной
  • .button--size-xl - размер кнопки на 100% больше стандартной
  • .button--fit - кнопка растягивается на всю ширину родительского блока

Прозрачность кнопок

  • .button--only-borders - показываются только границы кнопок, без заливки
  • .button-transparent - по умолчанию кнопка без границ и без фона, при наведении мышки появляется фон
  • .button-transparent-full - кнопка без границ и без фона, при наведении ничего не происходит

Группы кнопок

Если кнопки имеют схожее поведение, то можно объединить их в единый блок с классом .button-group

Модификаторы

Скругление крайних кнопок

Можно указать степень закругления крайней левой и крайней правой кнопки

  • .button-group--smooth - скругление, аналогично button--smooth
  • .button-group--rounded - закругленные края, аналогично button--rounded
  • .button-group--no-smooth - без скругления

Сообщения

Сообщения оборачиваются в блок с классом alert. Он состоит из двух компонентов: заголовок alert__header и контент alert__content. В заголовке может быть добавлена кнопка закрытия alert__header__close-btn. Если у вас подключен файл altrone.js, то кнопка будет скрывать ваше сообщение.

Списки

Чтобы создать список, нужно разметить элементы <div> или <li> внутри контейнера <div> или <ul>, к которому применен класс .list

Элемент списка. Можно использовать как обычный <li>, так и <div class="list__item">

Иконки и картинки

  • .list--icon - модификатор указывает, что в списке могут находиться иконки. Иконки должны быть в теге <i> или обозначены классом .icon
  • .list--image - модификатор указывает, что в списке могут находиться изображения. Изображения должны быть в теге <img> или обозначены классом .image

Анимация

  • .list--animation-slide - при наведении курсора текст сдвигается вправо
  • .list--animation-underline - при наведении курсора текст подчеркивается
  • .list--animation-invert - при наведении курсора фон заливается акцентным цветом (зависит от модификатора .list-color-*)
  • .list--animation-zoom - при наведении курсора весь контент увеличивается на 50%
  • .list--animation-border - при наведении появляется граница слева. Если используется .list--orientation-horizontal, то граница появляется снизу
  • .list--animation-mark-* - при наведении появляется маркировка у списка (вместо * нужно указать circle, disc или square). Работает только в том случае, если вы использовали <ul> и <li>
